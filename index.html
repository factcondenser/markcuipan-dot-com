Some of the things I've made...



Optimize Your Grow
Dec '17

Angular frontend. Julia backend. Built by Ian Atkins, who was having an issue with requests timing out while the server was busy solving a gnarly diffEq. I hacked together a solution and learned a ton about task queuing and Heroku's process model. The result is this proof-of-concept for a site that will help farmers and scientists alike optimize their crop yields.

Game of Fifteen
Nov '17

Originally implemented in C for a CS50x assignment, this is the second iteration of my take on the classic 15-puzzle. To help me retain what I was learning better, I wrote every line from scratch using JavaScript. What I learned is that JS is a huge pain in the arse when used incorrectly. I keep hearing rumors about the good parts of JS. I've only gotten good enough to sort of see these good parts myself, but I'm working on it! I made this before I started learning about responsive web design, but that will be the next thing I implement. For now, the game is best played on a computer with a keyboard.


Breakout
Oct '17

This little project served as my introduction to the functional programming paradigm. Having only really studied object-oriented programming up to this point, my encounter with this "new" paradigm sent me down a wiki-hole that eventually led to this video on Lisp. Learning about Lisp macros made reading the Julia code from Optimize Your Grow (see above) easier. I wrote a sillier version of this app using the Phaser game framework just for fun.

Twitter Clone

A much-simplified version of the actual Twitter, this came as the result of months spent going through Michael Hartl's Ruby on Rails Tutorial. I learned the Model-View-Controller pattern and how to do version control with Git. This project also introduced me to SQL, unit tests, integration tests, Bootstrap, and Sass. Kanjure
Summer '15

A mobile app for bringing up the pronunciation and definition of a Chinese character by simply touching it with your finger. I've been wanting to build this ever since I moved to Hong Kong post-college and found translating the texts of my cousins with something like the free tier of Pleco (and having to constantly flip between app activities) to be sufficiently annoying. This is a work in progress, since I'm still in the process of learning enough Android and Swift to build it out. Once I'm done, I'll put it up on both Google Play and the App Store.


Nannon Bot
Fall '14

I wrote the code for this as part of the Introduction to Artificial Intelligence course that I took in college. I actually wrote several Nannon players implementing various algorithms, but the most successful one used a Bayesian network as its model. Our professor pitted everyone's players against each other in an intra-class nannon bot tournament. My Bayes net player placed 12 out of 37. I want to update this project by creating an interface that allows a human player to play against the AI.

qPCR Concentrations Calculator
Summer '14

There's a reason I traded wet labs for DRY computing (See what I did there?). Talk about repetitive! During my time as a research assistant, I spent way too much time punching numbers into an Excel spreadsheet and running simple calculations on them. Enter this Python script. It takes a CSV file of measurements from a Qubit device and spits out the necessary volumes of various substances that need to be mixed in test tubes in order to synthesize cDNA. It was my first time writing Python and, coming from Java, I really didn't miss the semicolons.
